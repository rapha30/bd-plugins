# BD Plugins

Plugins para BetterDiscord feitos pelo Rapha.

## Plugins

### GlobalSearch
Busca mensagens em todos os servidores (ou servidores selecionados) de uma vez. Resultados ordenados do mais recente ao mais antigo.

**Features:** busca global, historico, pause/resume, filtro por canal, auto-refresh, modos de visualizacao (compact/traditional/detailed).

### ServerActivity
Mostra metricas de atividade de todos os servidores para ajudar a decidir quais manter ou sair. Zero chamadas de API - usa dados locais do cliente.

**Features:** tabela com metricas (ultima atividade, membros online, nao lidos, mencoes), cores de saude, filtro por pasta de servidores, botao para sair do servidor.

## Como instalar

### 1. Instalar BetterDiscord
Se ainda nao tem, baixe e instale: https://betterdiscord.app/

### 2. Baixar os plugins
Clique no link do plugin que quer instalar:

- **[GlobalSearch.plugin.js](https://raw.githubusercontent.com/rapha30/bd-plugins/master/GlobalSearch.plugin.js)** (clique com botao direito > "Salvar link como...")
- **[ServerActivity.plugin.js](https://raw.githubusercontent.com/rapha30/bd-plugins/master/ServerActivity.plugin.js)** (clique com botao direito > "Salvar link como...")

### 3. Colocar na pasta de plugins
Mova o arquivo `.plugin.js` para a pasta de plugins do BetterDiscord:

**Windows:** `%appdata%\BetterDiscord\plugins\`

Para abrir essa pasta rapidamente: aperte `Win+R`, cole o caminho acima e aperte Enter.

### 4. Ativar o plugin
No Discord, va em **Configuracoes > BetterDiscord > Plugins** e ative o plugin.

## Atualizacoes
Os plugins se atualizam automaticamente pelo BetterDiscord. Quando uma versao nova for lancada, voce recebe a atualizacao sem precisar fazer nada.
